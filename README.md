## Overview

This project provides a comprehensive field strategy for a 4-person door-knocking team targeting Okehampton, Devon (EX20) and surrounding areas for Hometree Finance 25-year hybrid solar + battery lease sign-ups.

### Key Features

- **Interactive Maps** - Leaflet.js maps with drive-time zones and target areas
- **Data Visualizations** - Plotly.js charts for electricity consumption, zone scoring, conversion funnels
- **Complete Field Guide** - Daily schedules, pitch scripts, objection handling
- **Postcode Analysis** - Ranked priority zones based on consumption, ownership, saturation
